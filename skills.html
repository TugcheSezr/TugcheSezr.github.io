Wat kan ik best met programmeren?

Wat betekent HTML?

HTML is de afkorting van Hyper Text Markup Language. 'Hyper Text' betekent tekst met links naar andere informatie. 'Markup Language' betekent opmaaktaal.

HTML 90%

Wat betekent CSS?

CSS is een opmaaktaal die de stijl van HTML-elementen bepaalt, zoals kleuren, lettertypen en lay-out. Het maakt je site mooier en makkelijker te onderhouden.

CSS 90%

Wat betekent JavaScript?

JavaScript is een programmeertaal die je website interactief maakt. Denk aan sliders, formulieren, of spelletjes zoals XOX.

JavaScript 85%
